# VectorBT Strategy Comparison  
EMA Crossover vs RSI vs Buy & Hold  

This project demonstrates quantitative trading backtesting in Python using [VectorBT](https://github.com/polakowo/vectorbt) and Plotly.  
It compares two classic technical strategies (EMA Crossover, RSI) against a Buy & Hold benchmark.  

**Key Features**
- Download financial data from Yahoo Finance  
- Backtest strategies with realistic fees  
- Compute key performance metrics  
- Visualize equity curves and drawdowns interactively  

**Tech Stack**
- Python  
- VectorBT  
- Plotly  
- Pandas  

**Output Example**
- `strategy_summary.csv`: Comparison table of total return, drawdown, and risk metrics  
- Interactive equity and drawdown plots
